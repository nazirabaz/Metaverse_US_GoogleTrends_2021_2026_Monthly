# Metaverse Interest in the United States - Google Trends Dataset (2021–2026)

This repository contains the processed monthly dataset used in the research paper:

**"Beyond Hype and Collapse: Metaverse Interest in the United States via Google Trends with SEM-ANN-fsQCA"**

### Dataset Description

- **File Name**: `Metaverse_US_GoogleTrends_2021_2026_Monthly`

- **Time Period**: January 2021 to January 2026 (Monthly level)

- **Geographic Scope**: United States

- **Source**: Google Trends (Publicly available)

- **Data Type**: Normalized search interest (0–100 scale)

### Variables / Columns in the Dataset

The CSV file includes the following columns:

- `Date` – Month and Year (YYYY-MM format)

- `Metaverse_Interest` – Overall search interest for "metaverse" and "virtual reality"

- `Performance_Expectancy` – Keywords related to benefits, uses, and work in Metaverse

- `Social_Influence` – Keywords related to social activities, concerts, friends, and parties

- `Facilitating_Conditions` – Keywords related to hardware and accessibility (e.g., Apple Vision Pro, VR)

- `Cognitive_Dissonance` – Keyword related to negative aspects ("metaverse scam")

### How to Reproduce the Data

All raw data can be freely reproduced from [Google Trends](https://trends.google.com/trends/) using the following settings:

- **Region**: United States

- **Timeframe**: January 2021 – January 2026

- **Category**: Web Search

- **Keywords**: See Table 2 in the research paper for the exact list of keyword proxies used for each construct.

### Purpose of This Dataset

This dataset was created to support a multi-method empirical study combining:

- Partial Least Squares Structural Equation Modeling (**PLS-SEM**)

- Artificial Neural Network (**ANN**)

- Fuzzy-set Qualitative Comparative Analysis (**fsQCA**)

The study investigates the drivers of public interest in the Metaverse during its rise and collapse phase.
